# -Pursuit-of-Jade-Digital-Notes
A curated web archive and interactive companion site exploring the narrative architecture of 逐玉.

An elegant digital archive and web companion exploring the narrative architecture, character loyalty arcs, and thematic depth of *Pursuit of Jade*. 

https://raedwulfx.github.io/-Pursuit-of-Jade-Digital-Notes/

This project features a fully responsive single-page design utilizing high-end typography, glassmorphism UI components, and smooth interactive states.

## 🌟 Key Features

* **Dynamic Navigation:** Fully integrated client-side tab switching across four core modules: Themes, Loyalty, Episodes, and Quotes.
* **Dual-Theme Engine:** Seamless toggle between a signature Deep Forest Jade theme (Dark) and a refined Parchment theme (Light).
* **Curated Interface:** Responsive glassmorphism cards styled with Tailwind CSS, utilizing a sophisticated blend of *IM Fell English* and *Noto Serif SC* typography.
* **Atmospheric Polish:** Fluid CSS transitions, interactive hover enhancements, and a subtle floating background particle system.

## 🛠️ Built With

* **Markup & Structure:** HTML5
* **Styling & Layout:** Tailwind CSS (via CDN)
* **Interactivity & Logic:** Vanilla JavaScript (ES6+)
* **Typography:** Google Fonts (*IM Fell English*, *Noto Serif SC*)

## 🚀 Deployment

The site is optimized for static hosting and can be instantly served via GitHub Pages. 

To view the live environment locally, clone the repository and open `index.html` in any modern web browser.
